## البدء السريع

### المتطلبات الأساسية

- Node.js 16+ و npm أو yarn
- متصفح حديث يدعم ES6+
- (اختياري) مفتاح API من OpenAI للمساعد الذكي

### التثبيت السريع

```bash
# 1. استنساخ المشروع أو الانتقال للمجلد
cd noor

# 2. تثبيت الحزم
npm install

# 3. بدء خادم التطوير
npm run dev

# 4. افتح http://localhost:3000 في المتصفح
```

## المكونات الرئيسية

### 1. **المكونات التنقلية (Navigation)**

#### Navbar.jsx

- الشريط العلوي الثابت
- قائمة التنقل
- تبديل المظهر (فاتح/داكن)
- زر القائمة على الأجهزة المحمولة

### 2. **المكونات الذكية (AI)**

#### FloatingAssistant.jsx

- زر عائم في الزاوية السفلى اليسرى
- قابل للتفاعل والتحريك
- يفتح واجهة الدردشة

#### ChatWidget.jsx

- واجهة دردشة عائمة
- يعرض سجل المحادثة
- إدخال البيانات وإرسال الرسائل
- مؤشر الكتابة

### 3. **مكونات الواجهة (UI)**

#### Button.jsx

أزرار قابلة لإعادة الاستخدام مع خيارات متعددة

#### Card.jsx

بطاقات بتصميم Glassmorphism

#### Toast.jsx

إشعارات مؤقتة

#### ProgressBar.jsx

شريط التقدم العلوي

### 4. **الصفحات (Pages)**

#### HomePage - الصفحة الأولى

- قسم البطل (Hero) مع رسومات جميلة
- ميزات المنصة
- أزرار الإجراء (ابدأ التعلم، تحدث مع نور)

#### AboutPlatform - عن المنصة

- رسالة المنصة
- الرؤية والقيم
- الفوائد التعليمية

#### Dashboard - لوحة التحكم

- إحصائيات التعلم
- المواضيع المكتملة
- درجات الاختبارات
- الأهداف

#### TopicsExplorer - استكشاف المواضيع

- عرض جميع المواضيع بشكل بطاقات
- بحث وتصفية
- مستويات الصعوبة

#### TopicDetail - تفاصيل الموضوع

- محتوى الموضوع المفصل
- أقسام قابلة للتوسع (Accordion)
- أزرار تفاعل
- اسأل نور

#### Timeline - الخط الزمني

- أحداث تاريخية مهمة
- رسوم متحركة جميلة
- معلومات مفصلة

#### Quiz - الاختبارات

- أسئلة متعددة الخيارات
- تقييم فوري
- عرض النتائج والشرح

#### ChatSection - دردشة كاملة الشاشة

- واجهة دردشة متكاملة
- سجل محادثة كامل
- أسئلة مقترحة

#### Sources - المصادر والمراجع

- قائمة شاملة للمراجع
- تصنيفات مختلفة
- شروط الاستخدام

## الميزات

### ✨ الميزات الأساسية

1. **تصميم ثنائي اللغة (RTL)**
   - واجهة كاملة بالعربية
   - دعم النصوص من اليمين لليسار
   - خط Cairo العربي الجميل

2. **وضع فاتح/داكن**
   - تبديل سلس بين الأوضاع
   - توفير للبطارية
   - محفوظ في Local Storage

3. **المساعد الذكي "نور"**
   - يرد على الأسئلة عن السيرة
   - يشرح المواضيع بطريقة مبسطة
   - يقترح خطوات تالية
   - يدعم OpenAI API و Mock Responses

4. **تتبع التقدم**
   - حفظ المواضيع المكتملة
   - تتبع درجات الاختبارات
   - عرض الإحصائيات

5. **نظام الاختبارات المتقدم**
   - اختبارات متعددة الخيارات
   - تقدم فوري
   - شرح الإجابات
   - حفظ النتائج

6. **التصميم الحديث**
   - Glassmorphism
   - تدرجات ناعمة
   - رسوم متحركة سلسة
   - استجابة كاملة على جميع الأجهزة

## التطوير والتحديث

### باب الملفات الرئيسية

**src/constants/data.js** - البيانات الثابتة:

- المواضيع (TOPICS)
- أحداث التايم لاين (TIMELINE_EVENTS)
- أسئلة الاختبارات (QUIZ_QUESTIONS)
- المصادر (SOURCES)

**src/services/AIService.js** - خدمة الذكاء الاصطناعي:

- الاتصال بـ OpenAI API
- Mock Responses للاختبار
- معالجة الأسئلة

### إضافة موضوع جديد

1. افتح `src/constants/data.js`
2. أضف موضوع إلى `TOPICS`:

```javascript
{
  id: 6,
  title: "موضوع جديد",
  description: "وصف...",
  icon: "📖",
  duration: "50 دقيقة",
  difficulty: "متوسط",
  progress: 0,
  sections: [
    {
      id: 1,
      title: "القسم الأول",
      content: "المحتوى...",
      timestamp: "التاريخ",
      expanded: false
    }
  ]
}
```

3. أضف اختبار إلى `QUIZ_QUESTIONS[6]`

### تخصيص الألوان

عدّل `tailwind.config.js`:

```javascript
colors: {
  primary: { ... },
  secondary: { ... }
}
```

## استكشاف الأخطاء

### الصفحة بيضاء

- تحقق من أخطاء الكونسول (F12)
- أعد تشغيل الخادم: `npm run dev`

### المساعد لا يرد

- لا توجد مشكلة، يستخدم Mock Responses
- أضف مفتاح OpenAI لتفعيل الاتصال الحقيقي

### البيانات لم تُحفظ

- تأكد من أن Local Storage مُفعّل
- تحقق من الكونسول من الأخطاء
