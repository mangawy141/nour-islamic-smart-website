# ⚡ البدء السريع

## التثبيت في 3 خطوات

```bash
# 1. تثبيت الحزم
npm install

# 2. نسخ ملف البيئة
cp .env.example .env

# 3. تشغيل الخادم
npm run dev
```

ثم افتح **http://localhost:3000** ✅

---

## الأوامر الرئيسية

| الأمر             | الوصف                  |
| ----------------- | ---------------------- |
| `npm run dev`     | تشغيل خادم التطوير     |
| `npm run build`   | بناء الإصدار النهائي   |
| `npm run preview` | معاينة الإصدار النهائي |

## هيكل سريع

```
noor/
├── src/
│   ├── components/    # مكونات React
│   ├── pages/         # الصفحات
│   ├── services/      # الخدمات (AI)
│   ├── constants/     # البيانات
│   └── App.jsx        # المكون الرئيسي
├── package.json       # الحزم
├── tailwind.config.js # إعدادات Tailwind
└── README.md          # التوثيق كامل
```

---

## إضافة مفتاح OpenAI (اختياري)

1. توجه إلى: https://platform.openai.com/api-keys
2. أنسخ المفتاح
3. الصق في `.env`:

```env
REACT_APP_OPENAI_API_KEY=sk-your-key
```

بدون مفتاح، النظام سيستخدم إجابات وهمية تلقائياً ✨

## استخدام المساعد الذكي

1. اضغط على ✨ في الزاوية السفلى اليسرى
2. اكتب سؤالك
3. اضغط Enter

أمثلة:

- "من هو النبي محمد؟"
- "ما دروس السيرة؟"
- "شرح لي غزوة بدر"
